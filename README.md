# AdventureWorks-Analysis
In this project, I analyzed sales, product, territory, and marketing campaign data from AdventureWorks to identify the most profitable markets and support strategic marketing investment decisions.

## Key Highlights

- Built unified datasets from multiple sources using SQL joins  
- Performed data cleaning and validation with COALESCE  
- Aggregated metrics by territory and country  
- Calculated financial KPIs (Margin and ROI)  
- Identified high-performing markets and optimization opportunities  
- Translated results into business recommendations  

## Skills Demonstrated

- SQL  
- Data Cleaning  
- Multi-table Joins  
- KPI Analysis  
- Financial Analytics  
- Business Intelligence  
- ROI Modeling  
- Data Storytelling  

## Outcome

Delivered a territory-level financial performance report that enables executives to prioritize profitable markets, optimize marketing spend, and improve overall business efficiency.
